# 01 - 系统概述

## 产品定义

**书籍推荐系统 (BookRecommend)** 是一个 AI Agent 驱动的智能书籍推荐平台。用户通过自然语言与 AI 进行多轮对话，系统理解用户的阅读偏好和需求，结合真实的书籍数据库，提供个性化、有依据的书籍推荐。

## 核心功能

### 1. 智能对话推荐
- 用户可与 AI 进行多轮自然语言对话
- AI 通过提问了解用户的阅读偏好（题材、难度、长度、语言等）
- 基于对话上下文理解用户意图

### 2. 关键词书籍搜索
- 用户输入关键词（书名、作者、题材、ISBN 等）
- 系统通过 Open Library API 检索真实书籍数据
- 返回书籍封面、作者、简介、出版信息

### 3. 偏好分析与记忆
- 短期记忆：保存当前会话的对话历史
- 长期记忆：通过 ChromaDB 向量存储用户偏好
- 支持跨会话偏好追踪

### 4. 多步骤推理推荐
- LangGraph 驱动的推荐工作流
- 理解意图 → 搜索书籍 → 分析偏好 → 生成推荐
- 当信息不足时主动询问用户

## 用户角色

| 角色 | 描述 |
|------|------|
| 普通用户 | 搜索书籍、与 AI 对话获取推荐 |

## 技术约束

- 系统为本地运行的单用户原型
- 依赖 Anthropic API（需要有效的 API Key）
- ChromaDB 嵌入式运行，数据存储在本地
- Open Library API 为公开免费接口
